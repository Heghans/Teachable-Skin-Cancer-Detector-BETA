# Teachable-Skin-Cancer-Detector-BETA
Teachable Skin Cancer Detector BETA es un prototipo experimental desarrollado con Teachable Machine y tecnologías web modernas, diseñado para demostrar cómo la inteligencia artificial puede asistir en la detección temprana de posibles anomalías en la piel. El sistema permite cargar una imagen y obtener una clasificación preliminar de esta.
